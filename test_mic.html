How to use

  1. Make sure the backend is running: python manage.py runserver 8000
  2. Open this file in your browser (just double-click it or drag into Chrome)
  3. Pick the patient's language from the dropdown
  4. Click "Start Recording" and speak as the patient (in their language)
  5. Click "Stop Recording" when done — the audio gets sent to the backend
  6. Watch the translation + medical flags appear
  7. Type a doctor response in the text box and hit Enter — it gets translated + spoken back
